# 🫀 Cardiovascular Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts the risk of cardiovascular disease using patient health parameters and a Machine Learning model deployed using Flask.

## ⚙️ Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS

## 🤖 ML Model
- Random Forest Classifier
- Dataset: Cardiovascular Disease Dataset (70,000 records)
- Accuracy: ~74%

## 📊 Features
- Age
- Gender
- Height
- Weight
- Blood Pressure
- Cholesterol
- Glucose
- Smoking, Alcohol, Activity

pip install -r requirements.txt
